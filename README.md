# Student-Management-System

路 Implemented the API endpoints for school management system with CRUD executions using Java Spring
MVC.
路 Utilized the JPA to enable the fast interaction between database and model, managing and querying entities.
路 Implemented a mapper to realize complex queries using MyBatis, facilitated the dynamic SQL query including
the fuzzy query.
路 Used Shiro framework to build the security management for the system, providing authentication, authoriza-
tion, and session management, securing the application.
